# DeerFlow 设计启示录（适用于 agent-core）

## 概述
DeerFlow 2.0 是一个超级智能体框架，在多 Agent 协作、技能系统、意图分解和上下文管理方面有诸多值得借鉴的设计。以下总结可帮助我们完善 agent-core 的对应模块。

## 设计对比与启示

| 模块 | DeerFlow 实现 | 对 agent-core 的启示 |
|------|---------------|----------------------|
| **多 Agent 协作** | 主代理动态生成子代理，每个子代理拥有独立上下文 | 完善 `replication` 模块，实现动态子代理生成和独立上下文管理 |
| **技能系统** | 用 Markdown 定义技能，支持按需加载 | 强化 `market` 模块，支持技能描述标准化、动态发现和按需加载 |
| **意图分解** | 将复杂任务拆解为 DAG（有向无环图） | 增强 `workflow` 模块，加入自然语言解析层，自动将用户意图转化为任务 DAG |
| **上下文管理** | 渐进式技能加载 + 摘要中间件 | 集成到 `storage` 模块，实现长会话记忆、上下文裁剪和摘要生成 |

## 下一步行动
- 将这些启示融入 `replication`、`market`、`workflow`、`storage` 模块的设计文档中。
- 后续可深入体验 DeerFlow，记录更多具体实现细节，用于指导开发。
